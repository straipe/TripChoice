# TripChoice

### 📌 프로젝트 설명  

## 🚀 기능  
-  [O] 국내 관광지 리스트 조회
-  [O] 여행 플랜 등록, 조회, 수정, 삭제
-  [O] 인기 관광지 등록, 조회
-  [O] 공지사항 등록, 조회, 수정, 삭제
-  [O] 날씨 정보 조회
-  [-] 현재 주소 기반 관광지 키워드 추천(GPT 활용)

## 🛠 설치 및 실행 방법  
```bash
# 1. 저장소 클론
git clone https://github.com/straipe/TripChoice.git

# 2. 디렉토리 이동
cd TripChoice

# 3. 필요한 패키지 설치
npm install  # 예시 (프로젝트에 맞게 수정)

# 4. 실행
npm start dev # 예시 (프로젝트에 맞게 수정)
